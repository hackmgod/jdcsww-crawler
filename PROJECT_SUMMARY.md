## ⚠️ 混合动力车辆说明

### 当前情况

已测试的参数代码:
- `rylx='C'` → 纯电动 ✅ (每批约30辆)
- `rylx='H'` → 0辆 (参数可能不正确)
- `rylx=''` (空) → 与rylx='C'相同

### 可能的原因

1. **参数代码未知**
   - 混合动力可能使用不同的参数名(不是rylx)
   - 或者使用不同的值(不是'H')

2. **查询方式不同**
   - 可能需要在网站选择不同的查询选项
   - URL结构可能不同

3. **数据分布**
   - 近期批次(401-403)可能没有混合动力
   - 需要检查更早的批次

### 解决方案建议

**方案A: 手动确认参数 (推荐)**
```
1. 访问 https://www.jdcsww.com/qcggs
2. 在网页上选择"混合动力"选项
3. 查看浏览器地址栏的URL参数
4. 将正确的参数添加到代码中
```

**方案B: 检查早期批次**
```
1. 早期批次(如300-350)可能有更多燃料类型
2. 运行 find_hybrids.py 检查多个批次
3. 找到有混合动力的批次
4. 分析该批次的URL参数
```

**方案C: 联系官方**

