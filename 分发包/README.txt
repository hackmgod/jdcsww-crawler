# 机动车环保网数据爬虫 - EXE版本

## 📦 文件说明

- **JDCCrawler** - 主程序（可执行文件）
- **EXE使用说明.txt** - 详细使用说明
- **README.txt** - 本文件

---

## 🚀 快速开始

### 第一步：运行程序

双击 `JDCCrawler` 即可启动程序

### 第二步：选择模式

程序会显示以下选项：

```
1. 完整模式 - 爬取所有批次+详情页 (82个字段，非常慢)
2. 快速模式 - 仅爬取列表页 (7个字段，快速)
3. 测试模式 - 爬取单个批次测试 (385批次)
4. 继续上次中断的任务
5. 仅导出已有数据为Excel
```

**首次使用建议选择 "3. 测试模式"**

---

## 📂 数据文件位置

程序运行后，**数据会保存在程序同目录的 `data` 文件夹**：

```
JDCCrawler所在的文件夹/
├── JDCCrawler          ← 程序本身
└── data/               ← 所有数据在这里（自动创建）
    └── production_enhanced/
        ├── vehicles_complete_all.xlsx   ← 🎯 你需要的表格
        ├── all_vehicles_complete.json   ← JSON数据
        └── crawler_*.log                ← 运行日志
```

## ⚡ 运行模式说明

### 模式1：完整模式
- **字段数**: 82个（含详情页）
- **时间**: 约3-5天（403批次 × 2种燃料类型）
- **适合**: 需要完整数据
- **注意**: 需要确认才能开始

### 模式2：快速模式
- **字段数**: 7个（仅列表页）
- **时间**: 约1-2小时
- **适合**: 快速获取基本信息

### 模式3：测试模式
- **字段数**: 82个
- **时间**: 约5分钟
- **适合**: 首次测试

### 模式4：继续上次
- 支持断点续传
- 从上次中断位置继续
- 不会重复爬取

### 模式5：仅导出
- 将已爬取数据导出为Excel
- 不进行网络爬取

## ✅ 功能特性

## 📋 数据字段说明

### 基本信息（7个）
- 批次、燃料类型、公告编号、公告型号
- 车辆名称、生产企业、详情链接

### 生产企业信息（24个）
- 中文品牌、企业名称、生产地址、注册地址
- 法人代表、电话、传真、邮箱等

### 三电参数（13个）
- **电芯信息**: 电芯类型、电压、容量、生产企业
- **电池信息**: 电池容量、电量、BMS、组合方式
- **电机信息**: 电机型号、功率、转速、生产企业
- **电控系统**: 电控供应商、充电机、DC/DC等

### 车辆参数（若干）
- 车辆总质量、外形尺寸、最高车速等

---

## ⚠️ 注意事项

1. **运行时间**: 完整模式需要数天，请确保电脑不会休眠或断电

2. **网络连接**: 需要稳定的网络连接

3. **防拦截**: 程序已内置防拦截策略（延迟、休息等）

4. **数据安全**: 每批次自动保存，意外关闭不会丢失数据

5. **断点续传**: 中断后重新运行，选择"4. 继续上次"即可

## 🔧 常见问题

### Q: data文件夹在哪里？
A: 就在JDCCrawler程序的同一目录下

### Q: 如何打开Excel文件？
A: 双击 `data/production_enhanced/vehicles_complete_all.xlsx`

### Q: 程序运行很慢，正常吗？
A: 正常。访问每个车辆详情页需要等待几秒，避免被拦截

### Q: 程序中断了怎么办？
A: 重新运行，选择"4. 继续上次"即可

### Q: 可以复制到其他电脑使用吗？
A: 可以。JDCCrawler是绿色软件，可以直接复制使用
