# 汽车公告数据爬虫

司机宝宝网 (jdcsww.com) 汽车公告数据采集工具

## 功能特点

- ✅ 支持按公告批次查询
- ✅ 支持纯电动车型筛选
- ✅ 列表页数据采集
- ✅ 详情页数据采集（带反爬虫处理）
- ✅ JSON 格式数据保存
- ✅ 随机延迟避免封禁

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 基础用法

```bash
python jdcsww_crawler.py
```

### 自定义配置

编辑 `config.py` 文件：

```python
CRAWL_CONFIG = {
    'start_batch': 403,      # 起始批次
    'end_batch': 400,        # 结束批次（None表示只爬取start_batch）
    'vehicle_type': 1,       # 车辆类型 (1=纯电动)
    'fuel_type': 'C',        # 燃料类型 (C=纯电动)
    'fetch_detail': True,    # 是否获取详情页
    'detail_limit': 5,       # 详情页爬取数量
    'delay_min': 2,          # 最小延迟（秒）
    'delay_max': 5,          # 最大延迟（秒）
}
```

## 数据说明

### 列表数据字段

- 车辆名称
- 公告型号
- 发布时间
- 公告批次
- 额定载质量
- 轴数
- 额定载客
- 燃料类型
- 外形尺寸
- 货箱尺寸
- 生产企业
- 详情链接

### 详情数据字段

包含列表数据所有字段，额外包含：

- 车辆类型
- 制造国
- 牌照类型
- 产品号
- 目录序号
- 中文品牌
- 英文品牌
- 车型
- 企业名称
- 生产地址
- 注册地址
- 车辆总质量
- 整备质量
- 最高车速
- 等等...

## 输出文件

数据保存在 `data/` 目录下：

- `vehicle_list.json` - 列表数据
- `vehicle_details.json` - 详情数据
- `jdcsww_data_YYYYMMDD_HHMMSS.json` - 带时间戳的数据文件
