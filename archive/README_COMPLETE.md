## 🚀 使用方法

### 快速开始

```bash
# 进入项目目录
cd tram-inspection/jdcsww-crawler/

# 爬取单批次
python3 complete_data_crawler.py

# 查看数据
cat data/vehicle_complete_data_batch385.json | python3 -m json.tool
```

### 自定义批次

编辑 `complete_data_crawler.py`:

```python
crawler.run(
    batch=403,        # 修改批次号
    detail_limit=30   # 修改爬取数量
)
```

### 多批次爬取

```python
batches = [403, 402, 401, 400, 399]
for batch in batches:
    crawler.run(batch=batch, detail_limit=30)
    time.sleep(10)  # 批次间延迟
```

## 🎯 技术要点

### ✅ 成功突破反爬虫

**关键发现：**
1. ❌ 直接访问详情页 → 被拦截
2. ✅ 先访问列表页 → 再访问详情页 → 成功

**实现方法：**
```python
# 1. 访问列表页建立会话
list_html = fetch_with_session(list_url)

# 2. 使用列表页的 Referer 访问详情页
detail_html = fetch_with_session(detail_url, referer=list_url)
```

### 🔑 核心技术

- HTTP Cookie 管理
- Referer 伪造
- 会话保持
- HTML 解析
- 三电参数提取
