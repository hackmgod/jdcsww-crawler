## 🛡️ 方案2：安全爬虫（推荐）

### 特点
- ✅ 大延迟：8-15秒/次
- ✅ 随机延迟
- ✅ 智能休息：每50批次休息30秒
- ✅ 自动重试
- ✅ 封禁检测

### 使用方法

```bash
# 运行安全爬虫
python3 safe_crawler.py
```

### 自定义配置

编辑 `safe_crawler.py`：

```python
self.min_delay = 8.0      # 最小延迟（秒）- 改为5-10
self.max_delay = 15.0     # 最大延迟（秒）- 改为10-20
self.batch_rest = 50      # 每N批次休息
self.rest_time = 30       # 休息时间（秒）- 改为60
```

### 延迟配置建议

**保守策略**（不易被封）：
- 延迟：8-15秒
- 休息：每50批次休息30秒
- 预计时间：约1-2小时（100批次）

**平衡策略**（推荐）：
- 延迟：5-10秒
- 休息：每30批次休息20秒
- 预计时间：约30-60分钟（100批次）

**激进策略**（可能被封）：
- 延迟：3-5秒
- 休息：每100批次休息10秒
- 预计时间：约15-30分钟（100批次）

## 🌐 方案3：Selenium爬虫（最稳定）

### 特点
- ✅ 使用真实浏览器
- ✅ 模拟人类操作
- ✅ 最难被检测
- ✅ 支持JavaScript渲染

### 安装依赖

```bash
# 安装Selenium
pip3 install selenium

# 安装ChromeDriver（macOS）
brew install chromedriver

# 如果上面命令失败，手动下载
# https://chromedriver.chromium.org/
```

### 使用方法

```bash
# 有头模式（可以看到浏览器操作）
python3 selenium_crawler_v3.py

# 无头模式（后台运行）
python3 selenium_crawler_v3.py --headless
```

### 注意事项

1. **首次运行**：Chrome会显示"正在受到自动化测试软件的控制"，这是正常的
2. **延迟时间**：建议8-15秒/次
3. **资源占用**：CPU和内存占用较高

## ⚠️ 重要提示

### 避免被封禁的建议

1. ✅ **使用随机延迟**：5-15秒
2. ✅ **分批爬取**：不要一次爬太多
3. ✅ **分时段爬取**：早上、下午、晚上分多次
4. ✅ **智能休息**：每50批次休息30秒
5. ✅ **模拟人类**：使用Selenium
6. ❌ **不要**：连续快速请求
7. ❌ **不要**：固定频率请求
8. ❌ **不要**：一次爬取大量数据

### 封禁后的处理

1. **立即停止**爬取
2. **等待15分钟**或更换IP
3. **检查状态**：`python3 check_ip_status.py`
4. **使用更保守的策略**继续
