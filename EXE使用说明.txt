【快速使用】

1. 双击 JDCCrawler.exe 运行程序
2. 根据提示选择运行模式（推荐先用测试模式）
3. 等待爬取完成
4. 在程序同目录的 data 文件夹中查看结果


【数据文件位置】

程序运行后会在 JDCCrawler.exe 旁边生成 data 文件夹：

  你的文件夹/
  ├── JDCCrawler.exe           ← 程序本身
  └── data/                    ← 所有数据在这里（自动创建）
      └── production_enhanced/
          ├── vehicles_complete_all.xlsx   ← 🎯 你需要的表格
          ├── all_vehicles_complete.json   ← JSON数据
          └── crawler_*.log                ← 运行日志


【如何找到数据】

Windows用户：
1. 右键点击 JDCCrawler.exe
2. 选择"打开文件所在的位置"
3. 你会看到 data 文件夹
4. 打开 data → production_enhanced
5. 双击 vehicles_complete_all.xlsx 查看数据

或者：
直接在 JDCCrawler.exe 所在的文件夹中查找 data 文件夹


【运行模式说明】

程序启动后会让你选择：

  1. 完整模式 - 82个字段，需要3-5天
  2. 快速模式 - 7个字段，需要1-2小时
  3. 测试模式 - 测试单个批次，5分钟
  4. 继续上次 - 断点续传
  5. 仅导出 - 将已有数据导出为Excel

首次使用建议选择"3. 测试模式"


【注意事项】

1. 程序运行时间较长，请确保电脑不会休眠或断电

2. 程序支持断点续传：
   - 如果中途关闭，下次运行选择"4. 继续上次"
   - 已爬取的数据不会丢失

3. 数据实时保存：
   - 每爬取一批数据就会自动保存
   - 即使意外关闭，已爬取的数据也在

4. 可以随时查看：
   - 爬取过程中也可以打开 data 文件夹
   - 查看已经爬取的数据


【常见问题】

Q: data文件夹在哪里？
A: 就在 JDCCrawler.exe 的同一目录下

Q: 如何打开Excel文件？
A: 直接双击 vehicles_complete_all.xlsx 即可

Q: 程序运行很慢，正常吗？
A: 正常，访问每个车辆详情页需要等待几秒

Q: 程序中断了怎么办？
A: 重新运行，选择"4. 继续上次"即可

Q: 可以复制到其他电脑使用吗？
A: 可以，JDCCrawler.exe 可以直接复制使用
